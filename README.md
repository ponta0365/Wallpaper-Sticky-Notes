# 壁紙付箋アプリ (Wallpaper Sticky Notes)

デスクトップの壁紙に直接付箋を焼き込んで表示する、軽量でスタイリッシュなWindows用付箋アプリケーションです。
通常時は壁紙として表示されるためクリックを透過し、作業の邪魔になりません。編集したい時だけ最前面に透過操作レイヤー（編集モード）を表示して、直感的に付箋の追加、移動、サイズ調整、削除が行えます。

---

## 🚀 主な機能

1. **壁紙合成表示（超軽量・クリック透過）**
   * 付箋をWindowsの壁紙（デスクトップ背景）にダイレクトに合成・適用します。
   * 通常時は付箋に対するクリックが全てスルーされるため、デスクトップアイコンの選択やファイルのドラッグ操作を一切妨げません。
2. **クイックメモ入力（グローバルホットキー）**
   * `Ctrl + Alt + N` を押すと、画面中央に美しい半透明のクイック入力窓が表示され、即座にメモを入力・追加できます。日本語IME（確定時のEnterキー）にも最適化されています。
3. **操作可能な編集モード**
   * タスクトレイアイコンを右クリックし、「付箋を移動・編集する (編集モード)」をONにすると、最前面の透明な操作レイヤーが表示され、付箋のドラッグ移動やサイズ変更、ダブルクリックでの「外す（削除）」ボタンの表示、右クリックでの個別テキスト変更・カラー変更・リマインダー設定が行えます。
   * 表示モードを「オーバーレイ」にしている場合は、デスクトップ上の付箋を直接シングルクリックするだけで、自動で編集モードに入ることができます。
4. **リマインダー通知**
   * 付箋ごとに通知予定日時を設定でき、時間になるとWindowsネイティブのタスクトレイバルーンチップで通知を行います。
5. **多彩な自動整列ルール**
   * 新しい付箋を追加する際、既存の付箋と重ならないように自動配置します。
   * 配置ルールは「斜めにずらす（カスケード）」「縦に並べる」「横に並べる」「右端から縦に並べる」「左端から縦に並べる」から選択でき、画面端に達した場合は自動で折り返します。
6. **マルチモニター ＆ 高DPI対応**
   * 複数モニターや、ディスプレイごとに異なるDPIスケーリング（拡大率）が設定されている環境下でも、座標ズレを起こすことなく物理ピクセルに正確に壁紙を合成します。
7. **データ管理（インポート・エクスポート・初期化）**
   * 設定や付箋データを一括で JSON ファイルに出し入れできます。会社のスケジュールやタスクを新入社員に共有する際などに最適です。また、すべての付箋をワンクリックでクリアする初期化ボタンも備えています。

## 🛠️ セットアップと起動方法（Windows）

本アプリは Python 3.8 以上がインストールされている環境で動作します。

### 1. ライブラリのインストール
リポジトリのルートディレクトリにある `setup.bat` をダブルクリックして実行してください。
自動的に Python の仮想環境（`.venv`）が作成され、必要な外部ライブラリ（`PySide6`, `Pillow`）がインストールされます。

### 2. アプリの起動
セットアップ完了後、`run.bat` をダブルクリックして実行すると、アプリがタスクトレイに常駐して起動します（コマンドプロンプトの黒い窓は非表示で起動します）。
